# 长期关系、夫妻关系与修复

## 长期关系不能用追求套路

进入伴侣或夫妻关系后，核心从“制造吸引”转为：

- 需求表达。
- 冲突修复。
- 信任维护。
- 家务/金钱/家庭边界协商。
- 亲密和自由的平衡。

## 冲突分析

先分清冲突类型：

| 类型 | 例子 | 处理重点 |
|---|---|---|
| 事实冲突 | 谁做了什么 | 还原事实，不扩大人格攻击 |
| 感受冲突 | 被忽视、不被尊重 | 先接感受，再谈解决 |
| 需求冲突 | 陪伴、空间、金钱、性 | 讲清期待和可执行协议 |
| 信任冲突 | 隐瞒、暧昧、出轨 | 停止伤害、透明、重建周期 |
| 家庭冲突 | 父母、彩礼、育儿 | 建立小家庭边界 |

## 沟通模板

```text
当发生 XX 时，我的感受是 XX。
我真正需要的是 XX。
我不是要指责你，而是想和你一起把 XX 调整好。
你觉得我们可以先试试 XX 吗？
```

## 道歉与修复

有效修复包含：

- 具体承认行为。
- 理解对方受到的影响。
- 不逼对方马上原谅。
- 提出可验证改变。
- 后续持续一致。

## 信任重建

如果涉及严重背叛：

- 先停止造成伤害的行为。
- 给对方知情权和选择权。
- 明确未来边界和透明机制。
- 接受修复需要时间。
- 必要时建议线下专业咨询。

## 亲密下降

不要直接归因“不爱了”。可能是压力、身体状态、关系怨气、生活疲劳、羞耻感、沟通方式或吸引下降。

建议路径：

1. 降低指责。
2. 询问感受和压力。
3. 恢复非性亲密。
4. 重新安排共同体验。
5. 再谈性需求和边界。

## 退出关系

当存在暴力、长期羞辱、控制、严重欺骗、反复越界且无改变时，优先支持安全退出和求助，而不是教用户“挽回”。
