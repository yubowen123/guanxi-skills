# 吸引力与自我建设

## 基本判断

当用户长期“没人聊、聊不动、见光死”，优先看系统问题，不急着给话术：

- 渠道是否匹配。
- 展示面是否真实且有吸引点。
- 外形和照片是否干净。
- 生活是否有可分享内容。
- 表达是否只有索取，没有贡献。
- 目标是否和自身资源、城市、阶段匹配。

## 展示面

展示面不是炫富或包装假人设，而是让别人看见一个真实、有生活密度的人。

可展示：

- 工作中的投入和能力，但不要抱怨堆满。
- 运动、审美、旅行、阅读、做饭、朋友、宠物、城市生活。
- 具体小故事，比抽象自夸更有用。

避免：

- 全是自拍或全无本人。
- 炫耀消费、车表酒局。
- 负能量、攻击性别、前任阴影。
- 过度精修导致见面落差。

## 社交渠道

根据用户画像匹配：

- 时间少、目标婚恋：相亲、朋友介绍、严肃交友平台。
- 兴趣明确：活动、课程、社群、运动、展览。
- 表达能力强：社交软件、内容平台。
- 圈子小：先扩弱关系，不直接追单一对象。

## 聊天能力

核心不是“会套路”，而是：

- 会观察对方给的线索。
- 会分享自己的生活。
- 会把话题从信息带到感受。
- 会在对方低反馈时停手。
- 会在窗口出现时邀约。

## 个人策略

输出时给用户一个短期动作表：

```text
本周：修资料/照片/渠道。
本月：建立稳定认识新人的机制。
三个月：复盘聊天和见面转化率。
```

## 资源匹配

收入、资产、城市、职业只用于判断生活方式和匹配度，不用于羞辱或物化。若用户陷入“只有钱/脸才有用”的极端判断，拉回到可改变量：形象、渠道、表达、情绪稳定、生活质量、筛选标准。
