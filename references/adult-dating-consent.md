# 成人约会、性沟通与约炮

## 硬性边界

只处理成年人之间自愿、清醒、知情、可撤回同意的亲密或性关系。

拒绝提供：

- 灌酒、下药、利用醉酒或意识不清。
- 对未成年人或无法同意的人进行性推进。
- 诱骗、隐瞒关键事实、冒充身份。
- 无视拒绝、纠缠、偷拍、传播隐私。
- 利用职权、师生、上下级等权力差压迫对方。

## 成人约会的核心

约炮或短期关系不是“没有规则”，而是规则更要讲清楚：

- 双方意图：短期、开放、探索、恋爱可能性。
- 安全：避孕、性病检测、地点安全、紧急联系人。
- 隐私：不拍照、不传播、不炫耀。
- 同意：每一步都可以停止。
- 后续：是否联系、如何告别、是否排他。

## 意图表达

不要伪装成长期恋爱来换取性关系。可以表达暧昧和吸引，但要给对方选择空间。

低压表达：

```text
我对你有吸引，也挺享受我们之间的暧昧。但我不想让你有压力，你如果只想慢慢了解或不往这个方向走，都可以直接说。
```

短期关系表达：

```text
我现在更适合轻松、诚实的成人约会，不想用恋爱承诺包装自己。如果你的期待是稳定关系，我会尊重，也不耽误你。
```

## 亲密确认

语言确认是加分，不是破坏氛围。

```text
我想吻你，可以吗？
```

```text
我们可以继续，但任何时候你不舒服都直接说，我会停。
```

## 风险提醒

遇到以下情况建议停止推进：

- 对方犹豫、沉默、身体僵硬、回避。
- 对方喝醉、情绪崩溃、刚经历创伤。
- 双方关系存在明显权力差。
- 对方对关系期待和用户明显不一致。

## 输出方式

成人约会建议必须包含：

```text
意图是否透明：
同意是否清晰：
安全措施：
建议话术：
停止推进的信号：
```
